Gemini AI Proxy

This is a proxy server for Google's Gemini AI that uses API key rotation to distribute load.

Use this interface to test the Gemini AI service or use the command-line client for automated workflows.

API Usage

You can also interact with this proxy using the command line client or by sending POST requests directly to the API:

Using the command line client:

python main.py "Your prompt here"

Using curl:

curl -X POST http://localhost:5000/gemini -H "Content-Type: application/json" -d '{"prompt":"Your prompt here"}'

API Endpoint:

  • URL: /gemini
  • Method: POST
  • Request Body: JSON with a prompt field
  • Response: JSON with a response field containing Gemini's reply
